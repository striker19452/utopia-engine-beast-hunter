# 乌托邦引擎：怪兽猎人

**Utopia Engine: Beast Hunter** — 一款单人骰子桌游的电子版

原版设计：Nick Hayes (2013)

代码作者：**striker1945**

---

## 游戏简介

你是一名被诬陷的猎人 Mason，曾是王都的武器匠学徒，却因刺杀神王的莫须有罪名被永久烙上刺客印记。你逃到了南方山谷中的一个小村庄，村庄正遭受恐怖巨兽的袭击。村中长老同意收留你，条件是你必须在 **14天** 内击败三只恐怖巨兽。

与此同时，王都的烈星军团正在南下，14天后将抵达山谷。如果他们发现你藏身于此，全村都将遭殃。

---

## 游戏目标

在14天内击败三只恐怖巨兽，赢得村庄长老的信任。

**胜利条件**：击败全部三只恐怖巨兽

**失败条件**（任一）：
- 生命值（HP）降为 0 — 你死了
- 村民怀疑值达到上限 — 你被赶出村庄
- 时间耗尽（划掉时间轨道最后一天）— 烈星军团到达

## 界面说明

### 顶部状态栏

| 元素 | 说明 |
|------|------|
| **HP** | 生命值，初始 10/10 |
| **决心** | 决心点，用于掷骰后 ±1 修正 |
| **怀疑** | 村民怀疑值，18格。满则游戏结束 |
| **天数** | 当前天数 / 总共14天 |

### 左侧：时间轨道

显示14天的进度。特殊标记：
- **E** = 事件日（第1、4、7、10、13天）
- **!** = 巨兽袭击日（第3、6、9、12天）
- 💀 = 最后一天

### 中央：三大区域

| 区域 | 普通材料 | 稀有材料 |
|------|----------|----------|
| 海尔比尔德峰 (Halebeard Peak) | 石头 | 白银 |
| 海岸洞穴 (Coastal Caverns) | 绳索 | 磁石 |
| 裂痕之地 (The Scar) | 焦油 | 黄铁矿 |

每个区域有 6 个搜索格，每个搜索格有 6 个空位。

### 右侧：三座防御塔

南方塔、西南塔、东方塔。三座塔楼形状各不相同，按各自塔形中的空格建造。

### 底部

- **长老审批**：三位长老，击败巨兽或建完三塔可获得审批
- **材料库存**：普通材料、稀有材料、怪物材料；每种材料最多保存 **4** 个，超出的材料弃置
- **村民小屋**：共9间。巨兽袭击方向没有完成的防御塔时摧毁1间；小屋被毁会立即增加怀疑，并会提高之后每日默认怀疑
- **操作按钮**：搜索、建造、锻造、合成、休息等

## 每日行动

每天你必须选择 **一项** 行动：

### 1. 搜索

**流程**：
1. 选择一个区域
2. 掷2个骰子，将结果分别填入搜索格的6个空位中
3. 重复掷骰3次，填满该搜索格的全部6个空位
4. 计算两个3位数的差值（上方数字 - 下方数字），查搜索结果表
5. 继续下一个搜索格，每天最多完成 **3次搜索**（恶劣天气下为2次，有厚外套则恢复为3次）

一旦开始一个搜索格，就必须把该搜索格完成。若提前停止当天搜索，未使用的搜索次数会失去。

**快速搜索**：如果你在一个区域只搜索1次，可以花费第2次搜索移动到另一区域，并在新区域进行第3次搜索。若任一区域受恶劣天气影响，不能快速搜索；厚外套允许在恶劣天气区域快速搜索。

**巢穴已发现**：如果某区域已标记“巢穴已发现”，你可以花费一次搜索遭遇该区域恐怖巨兽。挑战恐怖巨兽和搜索该区域同等处理，都会占用当天的一次搜索；不能在一个已经开始的搜索格中途使用此能力。

**搜索结果表**：

| 总和 | 结果 |
|------|------|
| 100 ~ 555 | 遭遇怪物（查遭遇等级表） |
| 11 ~ 99 | 追踪巨兽 + 1普通材料 |
| 1 ~ 10 | 找到巢穴 + 1稀有材料 或 2普通材料 |
| 0 | 找到巢穴 + 伏击加成 |
| -1 ~ -555 | 遭遇怪物（查遭遇等级表） |

**结果说明**：
- **追踪巨兽 + 1普通材料**：填入该区域1个追踪圈；若3个追踪圈全满，标记“巢穴已发现”并获得1决心点。然后获得该区域1个普通材料。
- **找到巢穴 + 1稀有材料 或 2普通材料**：标记“巢穴已发现”，获得1决心点，并选择获得该区域1个稀有材料或2个普通材料。
- **找到巢穴 + 伏击加成**：首次发现该区域巢穴时，标记“巢穴已发现”并获得1决心点；若在该结果界面立即挑战该区域恐怖巨兽，获得伏击加成。若巢穴此前已发现，则本次不再触发新的巢穴发现或伏击入口。

### 2. 建造

花费 1 个普通材料开始建造防御塔。

掷2个骰子 → 按 **邻接规则** 将数字填入该塔形状中的空格：
- 数字只能放在：**空邻格** 或 **有相同数字的邻格** 旁边
- 邻格指上下左右（不含对角线）
- 无法放置 = **故障(X)**，需花 1 普通材料修补，并在任意空格选择 X 的位置；若没有普通材料可修补，建造失败并消耗一天
- 全部填满 = 塔建成
- 无故障完成 = **+1 决心点**

**三塔全建** → 选择一位长老获得审批奖励。

### 3. 锻造

消耗对应的稀有材料开始锻造。每件装备使用自己的1个精炼格；精炼格有3列，每列上方和下方各1个空位。

**流程**：
1. 掷2个骰子，将两个结果填入该装备精炼格的任意空位
2. 每当某一列上下两个空位都有数字时，计算该列：上方个位数 - 下方个位数
3. 若该列结果为负数，受到1点伤害，并擦除该列重新来过
4. 若结果为0或正数，将结果写入该列结果圈
5. 继续掷骰并填入，直到三个结果圈都有数字
6. 三个结果圈相加，得到精炼值
7. 此时可额外花费同类型稀有材料，每花费1个使精炼值 -1

**锻造结果**：
- 精炼值 4或以上：失败，消耗一天
- 精炼值 1-3：锻造标准装备，消耗一天
- 精炼值 0：锻造精工装备，获得1决心点，消耗一天

| 装备 | 需要 | 标准 | 精工 |
|------|------|-----------|-----------|
| 银甲 | 白银 | 所有怪物攻击范围-1 | 同标准 + 每次最多受1点伤害 |
| 探矿杖 | 磁石 | 找到普通材料时，改为找到该区域1稀有材料 | 同标准 + 每当找到普通或稀有材料时，额外找到1稀有材料 |
| 分解矛 | 黄铁矿 | 你的攻击范围+1 | 同标准 + 攻击范围内两个相同数都算暴击 |

### 4. 休息

恢复 2 HP，消耗一天。**怀疑 +1**。

## 合成物品

合成物品不是每日行动。你可以在任意时间合成物品，但不能在战斗中合成。每个物品需要1个普通材料和1个怪物材料；同一物品最多只能拥有1个。

| 物品 | 材料 | 效果 |
|------|------|------|
| 血饵 | 污血+石头 | 搜索前使用，立即遭遇该区域任意一种普通怪物 |
| 强力诱饵 | 油肉+绳索 | 搜索前使用，以1次搜索遭遇该区域恐怖巨兽；即使尚未发现巢穴也可使用 |
| 厚外套 | 兽皮+绳索 | 无限使用；恶劣天气下可搜索3次，并可在恶劣天气区域快速搜索 |
| 火焰盒 | 龙烟+焦油 | 战斗中立即造成2点伤害，可用于在怪物反击前击杀怪物 |
| 蟹甲 | 噩梦甲壳+石头 | 忽略最多2点伤害（2次使用） |
| 复活药 | 巨魔灰+绳索 | HP降为0时自动恢复3HP |
| 鹰图腾 | 圣羽+焦油 | 安装到已完成的塔上，该塔免疫巨兽攻击 |

## 战斗系统

遭遇怪物时进入战斗：

1. 掷2个骰子
2. 每个骰子如果在 **你的攻击范围** 内 → 对怪物造成1点伤害
3. 同一轮中，每个骰子如果也在 **怪物的攻击范围** 内 → 你受到1点伤害。本轮双方命中会一起结算，因此即使你在这一轮击杀怪物，本轮怪物命中仍会造成伤害
4. 重复直到一方HP归零

若同一轮结算后双方都归零，程序会先判定你击败了怪物/巨兽。

**暴击**：基础暴击只在战斗中掷出双6时发生，造成额外1点伤害并获得1决心点。如果你为了达成暴击花费过决心点，则不能因该暴击获得决心点。精工分解矛会把暴击条件改为：攻击范围内两个相同数都算暴击。

**伏击加成（Ambush）**：你的攻击范围 +1，并且忽略怪物在前两次战斗掷骰中造成的伤害。

**逃离战斗**：你可以尝试逃离任意遭遇。掷2个骰子，只要至少一个骰子为5或6，就成功逃离。如果失败，检查这些骰子是否落在怪物攻击范围内；若命中，怪物照常造成伤害。然后你可以继续尝试逃离或恢复战斗。逃离前已对怪物造成的伤害会保留到下次遭遇。

### 恐怖巨兽特性

每只恐怖巨兽有一个随机特性：

| 特性 | 效果 |
|------|------|
| 温顺 (Cowed) | 每轮只要你造成至少1点伤害，巨兽攻击范围上限-1 |
| 残忍 (Cruel) | 掷出1时造成重伤（永久-1 HP上限） |
| 狡猾 (Cunning) | 与该巨兽战斗的前2轮你无法造成伤害；若你拥有伏击加成，则取消此效果 |

## 事件系统

### 事件日（E日）

当你划掉带 **E** 的日期时，分别为4个事件各掷1个骰子。骰子结果决定事件影响的区域：1-2 = 海尔比尔德峰，3-4 = 海岸洞穴，5-6 = 裂痕之地。事件会持续到下一个事件日；不能花费决心点改变事件骰。

| 事件 | 效果 |
|------|------|
| 丰饶 (Abundance) | 搜索到材料时 +1 |
| 突然清醒 (Sudden Clarity) | 该区域你的攻击范围 +1 |
| 恶劣天气 (Foul Weather) | 该区域每天只能搜索 2 次 |
| 疯狂 (Madness) | 该区域所有怪物 +2 HP |

**特殊**：每有一个区域同时受到 **恶劣天气 + 疯狂** 影响，就触发一次正常的恐怖巨兽袭击；袭击区域仍像 ! 日一样重新掷骰决定，并会重掷已击败巨兽的区域。

### 巨兽袭击日（!日）

恐怖巨兽袭击村庄：
1. 掷1个骰子确定袭击来自哪个区域；若该区域恐怖巨兽已被击败，则重掷。恶劣天气+疯狂触发的额外袭击也使用这一步重新确定袭击区域
2. 再掷1个骰子确定来袭方向（见下表）；不能花费决心点改变该骰
3. 如果该方向有完成的塔 → 塔吸收攻击（见塔损伤表）
4. 如果没有塔 → 一间小屋被摧毁，并按当前已毁小屋数量增加怀疑
5. 袭击结束后，在该巨兽所属区域标记1个追踪圈

**小屋规则**：村庄有9间小屋。每次巨兽无塔阻挡进入村庄，划掉1间小屋，并立刻按“当前已毁小屋数 ×2”增加怀疑：第一间 +2，第二间 +4，第三间 +6，依此类推。已毁小屋还会计入每天默认怀疑（除休息、完成造塔、击败恐怖巨兽之外，均为 1 + 已毁小屋数）。

**防御塔阻挡规则**：巨兽来袭方向只会检查对应方向的塔：东方塔阻挡东（E），南方塔阻挡南（S），西南塔阻挡西南（SW）。只有已完成的塔可以阻挡；未完成、被摧毁或不存在的塔都视为没有塔，巨兽会进入村庄摧毁1间小屋。若对应方向有已完成的塔，巨兽攻击由塔吸收，不摧毁小屋；随后为该塔掷1骰并加上塔内已有X数量，按塔损伤表结算。装有鹰图腾的完成塔受到0伤害，仍然成功阻挡。

**接近方向表**：

| 骰子 | 海尔比尔德峰 | 海岸洞穴 | 裂痕之地 |
|------|-------------|----------|----------|
| 1 | 东 | 东 | 东 |
| 2 | 东 | 东 | 南 |
| 3 | 南 | 东 | 南 |
| 4 | 南 | 东 | 西南 |
| 5 | 西南 | 南 | 西南 |
| 6 | 西南 | 南 | 西南 |

**塔损伤表**：

| 骰子 | 结果 |
|------|------|
| 1 | 无伤害 |
| 2 | 1点伤害（填入1个X） |
| 3-5 | 2点伤害（填入2个X） |
| 6+ | 塔被摧毁 |

塔受损时，程序会在塔内尚未损坏的建造格自动加入X。塔伤害骰需要加上该塔已有X的数量。塔被摧毁时，擦除整座塔（塔形外的空白格不受影响）。

## 怀疑系统

每天结束时增加怀疑。怀疑总是在一天的最后处理，也就是在事件和恐怖巨兽袭击都结算完之后：

| 行动 | 怀疑增加 |
|------|----------|
| 休息 | 1（不加被毁小屋数） |
| 完成造塔 | 0 |
| 击败恐怖巨兽 | 0 |
| 其他情况 | 1 + 被毁小屋数 |

小屋被摧毁时：按当前已毁小屋数量 ×2 增加怀疑（包括刚被摧毁的那间）。例如第一间被毁时 +2，第二间被毁时 +4。

## 决心点

掷骰后可花费决心点 **±1** 调整单个骰子（不能低于1或高于6）。

**使用方式**：在搜索、建造、锻造、战斗和逃跑掷骰后，骰子旁会出现 `+ / -` 按钮。点击一次消耗1点决心，将对应骰子调整1点。调整后再放置骰子或点击结算。不能修改事件骰、巨兽袭击区域骰、巨兽来袭方向骰和塔损伤骰。

**暴击限制**：如果你在一次战斗掷骰中花过决心点，并因此形成暴击，本次暴击仍造成额外1点伤害，但不能获得暴击奖励的1点决心。

**获取方式**：
- 找到巢穴
- 暴击
- 完美建造（无故障）
- 精工锻造
- HP降到1
- 获得西波罗斯的审批

## 初始物品

游戏开始时你拥有3件物品（每件只能使用一次）：

| 物品 | 效果 |
|------|------|

| 光学扰乱器 | 战斗中点击“使用物品”，立即逃离战斗 |

## 操作指南

### 基本操作

- 点击 **搜索** → 选择区域 → 掷骰 → 点击空位填入数字
- 点击 **建造** → 选择塔 → 选择材料 → 掷骰 → 点击网格放置
- 点击 **锻造** → 选择装备 → 掷骰 → 点击网格放置
- 点击 **合成** → 选择物品 → 确认材料足够
- 点击 **休息** → 确认

### 调试模式

- 点击 **🛠 调试** 打开调试面板
- 可自由修改 HP、HP上限、决心、怀疑、怀疑上限、当前天数、小屋状态、每种材料数量、区域巢穴/追踪、巨兽击败状态和装备等级
- 调试面板会遵守材料上限：每种材料最多 4 个

### 存档

- 点击 **💾 保存** 保存当前进度到浏览器本地存储
- 点击 **📂 加载** 恢复上次保存的进度
